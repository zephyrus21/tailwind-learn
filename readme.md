# Scripts

To build the styles.css file in public

```
  yarn build:css

  #or

  npm run build:css
```

To create `tailwind.config.js` file

also add `--full` tag for the default config file.

```
  yarn tailwindcss init

  #or

  npx tailwindcss init
```
